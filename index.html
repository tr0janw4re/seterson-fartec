FETEC

O que é a FETEC?

A FETEC é um evento promovido pela Etec Darcy Pereira de Moraes (Itapetininga - SP) e conta com a participação dos Professores e Alunos para exposição de projetos, experimentos e apresentações. Este site foi desenvolvido pelos alunos do 2º ano do curso Ensino Médio integrado ao Técnico em Informática para Internet, de modo a proporcionar ao público uma "exposição virtual" dos Projetos e Jogos desenvolvidos no decorrer do ano.

Informática para Internet


O curso Informática para Internet prepara o aluno para o desenvolvimento de websites, aplicativos e-commerce e serviços relacionados, além de fornecer uma base sólida em tecnologias de programação, banco de dados e desenvolvimento web. O curso promove habilidades essenciais para a sociedade atual. Vem ser Etec!
